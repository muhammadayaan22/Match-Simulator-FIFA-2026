# ⚽ FIFA 2026 AI Oracle

An AI-powered football match prediction and simulation platform for the FIFA World Cup 2026, built on a custom-trained ML prediction engine and LLM-generated match commentary.

Pick any two of the 48 qualified nations and get a win-probability breakdown, predicted scoreline, expected goals, possession split, tactical radar, and AI-generated minute-by-minute commentary — all grounded in the same underlying prediction, not generated independently.

## Features

- **Match Simulator** — win/draw probabilities, predicted score, xG, possession, shots, radar chart comparison, predicted Man of the Match
- **AI Match Commentary** — minute-by-minute broadcast-style commentary generated by Claude (Anthropic API), consistent with the predicted scoreline
- **AI Analyst** — chatbot for football questions, grounded in live team stats (lightweight RAG)
- **Historical Insights** — full 48-team rankings, sub-score breakdowns, squad lists, and a transparent model benchmark comparison (Logistic Regression vs Random Forest vs XGBoost)
- **Home dashboard** — World Cup countdown, live rankings, featured matchups

## How the prediction engine works

1. **Dataset**: 48 World Cup 2026 qualified teams seeded with FIFA rank, Elo rating, attack/midfield/defense sub-scores, and recent form. 6,000 synthetic historical matches are generated from a strength-differential simulator with realistic variance (upsets included) to train the models.
2. **Training**: Three classifiers — Logistic Regression, Random Forest, and XGBoost — are trained on rank/Elo/sub-score/form differentials and compared on accuracy, log loss, and macro F1. The model with the **best-calibrated probabilities (lowest log loss)** is selected for production, since a win-probability product depends on calibration, not just raw accuracy.
3. **Live prediction**: Two small Random Forest regressors estimate each team's expected goals, which are combined with the classifier's outcome probabilities to produce a consistent scoreline.
4. **AI layer**: The predicted scoreline and stats are passed to Claude as structured context, so the generated commentary and analysis always agree with the simulation rather than inventing a separate narrative.

> **Note on data**: All team ratings and the historical match dataset are synthesized for this project (not pulled from a live sports API), tuned to track real-world football intuition. Swap in a live data source via `data/generate_teams.py` if you want this running on real, current FIFA ranking feeds.

## Setup

```bash
# 1. Clone and enter the project
git clone https://github.com/muhammadayaan22/fifa2026-ai-oracle.git
cd fifa2026-ai-oracle

# 2. Install dependencies
pip install -r requirements.txt

# 3. Generate data, build the database, and train the models
python data/generate_teams.py
python data/generate_matches.py
python database/build_db.py
python models/train_models.py

# 4. (Optional) Enable AI commentary & analysis
export ANTHROPIC_API_KEY="your-key-here"

# 5. Run the app
streamlit run app.py
```

The app runs fully without an API key — Match Simulator, Historical Insights, and the database all work standalone. The AI commentary and analyst pages fall back to deterministic templated output if `ANTHROPIC_API_KEY` isn't set, so the app is always demoable.

## Deployment

### Streamlit Cloud
1. Push this repo to GitHub.
2. Connect it at [share.streamlit.io](https://share.streamlit.io).
3. Set `ANTHROPIC_API_KEY` under app secrets.
4. Set the main file to `app.py`.

### Docker
```bash
docker build -t fifa2026-ai-oracle .
docker run -p 8501:8501 -e ANTHROPIC_API_KEY=your-key fifa2026-ai-oracle
```

## Model Benchmarks

| Model | Accuracy | Log Loss | F1 (macro) |
|---|---|---|---|
| Logistic Regression | ~60.8% | ~0.879 | ~0.448 |
| Random Forest | ~59.7% | ~0.883 | ~0.451 |
| XGBoost | ~56.9% | ~0.917 | ~0.448 |

(Three-way classification baseline is ~33%; football outcomes — especially draws — are inherently high-variance, so these numbers track real-world bookmaker-level uncertainty rather than indicating a tuning problem.)
